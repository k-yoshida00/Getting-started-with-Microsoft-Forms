# Microsoft 365とは
 Office 365をはじめとしたMIcrosoftの様々なアプリを、サブスクリプション契約でりようできる。


Word,Excel,Powerpointなどのアプリを、クラウド（インターネット）に接続して利用することで、常に最新のバージョンでアプリを利用できたり、社内の人と簡単にファイル共有を行うことができる。
